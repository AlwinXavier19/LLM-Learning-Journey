# 🧠 My Structured LLM Learning Journey

## 📌 About

This repository captures my journey of learning **Large Language Models (LLMs)** in a structured and intentional way.

The goal is not just to go through a course, but to build a clear understanding of how things actually work — and to make that understanding easy to revisit later.

---

## 🎯 Purpose

- Build strong fundamentals in **Transformers and LLMs**
- Turn learning into a **clear and reusable knowledge base**
- Make future revision **fast and frictionless**
- Track consistency and progress over time

Hugging Face LLM Course:

## 🧠 Learning Approach

Each concept is broken down with focus on:

- Why it exists  
- What problem it solves  
- How it works  
- When to use it  

This keeps the learning practical and grounded.
